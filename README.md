### Requirements

In the project directory, you can run:

- [Node.js 18.12.1](https://nodejs.org/ca/blog/release/v18.12.1/)
- [Npm 8.19.2](https://www.npmjs.com/package/npm/v/8.19.2)

### Installation

```bash
$ git clone https://github.com/scs0209/myBlog.git
$ cd myBlog
```

#### Backend

```
$ cd server
$ npm i
$ npm run dev
```

#### Frontend

```
$ cd myBlog
$ npm i
$ npm run start
```
